
The JoinFactory creates new join instances.

## Functions
### createJoin
```solidity
  function createJoin(
    address asset
  ) external returns (address)
```

Deploys a new join.

#### Parameters:
| Name | Type | Description                                                          |
| :--- | :--- | :------------------------------------------------------------------- |
|`asset` | address | Address of the asset token.

#### Return Values:
| Name                           | Type          | Description                                                                  |
| :----------------------------- | :------------ | :--------------------------------------------------------------------------- |
|`join`| address | The join address.
